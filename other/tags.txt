- NOTE: Some tags have opening and closing tags like <p></p>, while some are single tag like <br/>
Typical HTML page format is similar to this:
<!DOCTYPE html>
<html>
	<head>
		<style>
		</style>
	</head>
	<body>
		<header>
		</header>
		<main>
		</main>
		<footer>
		</footer>
	</body>
</html>
- <!DOCTYPE html> = Declares what version of HTML is being used
- <html> = Root HTML tag
- <head> = Contains metadata like the character set, title, Favicons, scripts & CSS
- <body> = Contains contents of HTML file
- <meta> = Contains metadata that can't be represented by other HTML
	- EXAMPLE:
		<meta charset="UTF=8">
- <title> = Contains title of webpage
	- EXAMPLE:
		<title>Webpage Title</title>
- <h1> to <h6> = Header = Contains headers of various sizes, put in descending order
	- EXAMPLE:
		<h1>This header is bigger than header 2</h1>
		<h2>This header is smaller than header 1</h2>
		<h6>This header is smaller than both</h6>
- <p> = Paragraph = Contains bodies of text, you can use stuff like \n or \r
	- EXAMPLE:
		<p>The Yu-Gi-Oh! Trading Card Game is a collectible card game developed and published by Konami.</p>
- <br/> = Break = Provides line breaks, adding extra space between above and below elements
	- EXAMPLE:
		<p>Text A</p>
		<br/>
		<p>Text B</p>
- <hr/> = Horizontal Rule = Provides straight line to break up content
	- EXAMPLE:
		<h1>HEADER</h1>
		<hr/>
		<p>TEXT</p>
- <big> and <small> = Makes text slightly larger and smaller respectively
	- EXAMPLE:
		<p>OH...<small>sorry...</small></p>
		<p>Street Fighter III: 2nd <big>IMPACT</big> is a fun game</p>
- <sub> and <sup> = Subscript and Superscript = Applies subscript and superscript to text
	- EXAMPLE:
		<p>Fun fact: 13<sup>2</sup> is 169
		<p>Another fun fact: the formula for ammonia is NH<sub>3</sub></p>
- <!--Comment--> = Adding comments, extra information that isn't shown on webpage
	- EXAMPLE:
		<!--You can't see this on the webpage.-->
- <style> = Defines CSS (Cascading Style Sheets) information
	- EXAMPLE:
		<style>
			p {
				color:green;
				background-color:black;
			}
		</style>
	- NOTE:
		- Colors can be found on https://www.w3schools.com/cssref/css_colors.php, or with hex codes
- <header> = Contains header text
- <main> = Contains main content of page
- <footer> = Contains conclusive info of page
- <nav> = Contains large blocks of navigation links
- <article> = Contains independent & self-contained content, articles are modular
- <section> = Contains sections of a document
- <aside> = Contains information tangentially related to main information
- <a> = Anchor = Used for linking to other pages, like so...
	-EXAMPLE:
		<a href="https://www.youtube.com">
	to load YouTube on the same tab the page was on. Loading it in a new tab can be 
	done like so...
	- EXAMPLE:
		<a href="https://www.youtube.com" target="_blank">
- <img> = Image = Contains an image, with the image being stored externally and the HTML
	file referencing it like so...
	- EXAMPLE:
		<img src="https://smashbros.com/assets_v2/img/fighter/snake/main.png">
		with attributes like src, width & height, alt for alt text (used for screenreaders
		or when the image doesn't load, etc
- <video> = Contains a video, with the video being stored externally, and the HTML file
	referencing it like so...
	- EXAMPLE:
		<video src="https://interactive-examples.mdn.mozilla.net/media/cc0-videos/flower.webm">
		with attributes like srcc, width & height, controls, loop and poster for a thumbnail
- <iframe> = Inline Frame = Embeds another page within the current HTML page
	- EXAMPLE:
		<iframe width="560" height="315" src="https://www.youtube.com/embed/9WIFaVcsnKs" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>
- <ul> = Unordered List = Contains an unordered list
- <ol> = Ordered List = Contains an ordered List
- <li> = List Item = Contains an item held within a list
	- EXAMPLE:
		<ul>
			<li>Element 1</li>
			<li>Element 2</li>
		</ul>
- <dl> = Description list = Contains a description list
- <dt> = Description title = Contains the title for a list item
- <dd>	= Description description = Contains the description for the list item
	- EXAMPLE:
		<dl>
			<dt>Item title</dt>
			<dd>Item description</dt>
			<dt>Another item</dt>
			<dd>Another item description</dt>
		</dl>
- <table> = Contains a table
- <tr> = Table Row = Contains multiple table cells displayed in a single row
- <td> = Table Data = A single table cell,
	colspan attribute changes how many columns it occupies
- <th> = Table Header = A single table cell stylised like a header
- <thead> = Table Head = Contains table header cells
- <tbody> = Table Body = Contains table body cells
- <caption> = Contains a table caption
	- EXAMPLE:
		<table>
			<caption>Table name</caption
			<thead>
				<tr>
					<th>Header 1</th>
					<th>Header 2</th>
					<th>Header 3</th>
				</tr>
			</thead>
			<tbody>
				<tr>
					<td>Cell 1</td>
					<td>Cell 2</td>
					<td>Cell 3</td>
					<td colspan="3">This cell occupies 3 columns</td>
				</tr>
			</tbody>
		</table>
